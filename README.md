# Neel Upadhyay Portfolio

Static portfolio site built to work well on GitHub Pages with no build step.

## Quick Start

Open `index.html` directly in a browser, or run:

```bash
npm start
```

Then visit `http://localhost:3000`.

## Publish on GitHub Pages

1. Push this repo to GitHub.
2. Open the repository settings.
3. Go to `Pages`.
4. Set the source to `Deploy from a branch`.
5. Choose your main branch and the `/ (root)` folder.
6. Save, then wait for GitHub Pages to publish the site.

Because this is a plain static site, there is no build setup to manage.
